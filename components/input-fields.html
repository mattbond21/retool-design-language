Input fields are used in forms to help users enter, select, and search. Input fields are normally found within a form but can also be part of a modal, global header, or as a filter option on a page.

Usage and behavior

  • Input fields should be shown on white backgrounds by default. If that is not possible, invert the background of the input field to be white so it will be visible on a neutral-200 background fill
  • Label text should indicate what sort of information the field requires.
  • Label text sits above the input area when user selects the field.
  • Label text shouldn’t be truncated. Keep it short, clear, and fully visible.
  • Label text shouldn’t take up multiple lines.
