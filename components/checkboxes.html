Checkboxes are used in forms to collect input from users. They can be used for a single item, or for a list of items where people can choose any number of options, including zero.

Checkboxes require the use of a button to apply the settings once they are selected. When used in a group, each checkbox is independent of all other checkboxes in the list; checking one box doesn’t uncheck the others.

Example

A checkbox control has two primary states: selected and unselected. If required, there’s an additional visual state for indeterminate.

  1. Group label: Conveys the purpose of a group of checkboxes.
  2. Selected: An active option in the list.
  3. Unselected: An inactive option in a list.

Content guidelines

  • Write checkbox labels that explicitly describe the option. They should be worded as a statement that the selected state makes true, and the unselected state makes false.
  • Always use sentence case unless capitalization is required for words used in a checkbox label. For example: I agree to the Terms & Conditions
