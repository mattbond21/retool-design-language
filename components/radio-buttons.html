Radio buttons allow users to select a mutually exclusive option from a given set. Radio buttons should be used when the user needs to see all available options, up to a maximum of five options. If there are more than five options in the list, you should use a dropdown menu instead.

Example

  1. Radio group label: Conveys the purpose of a group of radio buttons.
  2. Unselected: An inactive option in a list.
  3. Selected: The active option in the list.

Usage and behavior

Radio buttons should always be stacked vertically for easy scanning. The list options should be shown in a logical order. For example, the most likely to be selected options at the top, and least likely to be selected options at the bottom.
