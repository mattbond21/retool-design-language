We use round avatars to quickly identify users on the Retool platform. No other entity or container should be expressed in a circular shape to avoid confusion between a user and another object type.

Usage

Name Size Avatar Example usage
Small 24px
Typeahead menus, inline meta data in tables
Medium 32px
User profile dropdown menu, or when viewing a list of users in a table in a stacked format (i.e. user permissions)
Large 64px
Profile and account pages for users

Default user avatars

We provide users with a default avatar based on the letter of their first name and a colored background from our customization color palette. If they don't upload a profile avatar during sign up, one of these defaults will be automatically added to their account.
