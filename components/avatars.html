We use round avatars to quickly identify users on the Retool platform. No other entity or container should be expressed in a circular shape to avoid confusion between a user and another object type.

Usage

Name Size Avatar Example usage
Small 24px
Typeahead menus, meta data on card containers
Medium 32px
User profile dropdown menu, or when viewing a list of users in a table (i.e. class members)
Large 48px
Set page author details
X-Large 64px
Profile page headers

Default user avatars

We provide users with 20 default avatars to choose from in settings. If they don't upload a profile avatar during sign up, one of these defaults will be automatically added to their account.
