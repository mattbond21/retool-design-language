The Retool Design Language encourages consistency across platforms and screen sizes by using uniform elements and spacing.

By using our standard spacing, the user interface remains consistent, can be scaled easily, and uses rhythm to align elements correctly.

We utilize an 8px “soft grid” for base spacing and element sizes. The below spacing increments apply to both horizontal and vertical spacing between elements.

Exceptions

Due to the natural of the Retool UI, we require smaller units, and at times, units that fall between the 8px grid. We make an exception for using 4px as a half unit in specific circumstances, and at the specific sizes below.

It is important to use these spacing sizes as they are prescribed, and not to combine them. For example, 56px (combining 28px twice) should not be used even though it is divisible by 8px. Instead us a standard spacing size from the samples provided.
