We use shape in combination with other customizations, like color, spacing, and typography, to develop Retool’s visual design language. Rectangular shapes should always have rounded corners to help convey a friendly and approachable brand to users.

Scaling rounded corners

The unit value of a rounded corner should scale up or down depending on the size of the element or component that is displayed on screen. Generally speaking, the larger the component, the larger the rounded corner radius should be. This helps to maintain proportions for instances where components are nested inside of one another.

Examples

Small radius

We use the small radius on the smallest UI components such as buttons, badges, and tooltips.

Medium radius

We use the medium radius on UI components such as cards, toasts, and menus.

Large radius

We use the large radius on UI components such as the set preview, study mode containers, and modal dialogs.
