# Retool Design Language
The design system for Retool. WIP right now.

## Dependencies

- Ruby 3.0.1 (easily installed by rbenv)
- Jekyll 4.2
- git

## Commands and Hosting
Run `bundle exec jekyll serve --baseurl '/'` to develop locally.
